# Strategic Product Placement Analysis

## Project Overview

This Tableau project analyzes how product placement impacts sales performance in a retail environment.

## Key Insights

* Products placed at end-caps generate higher sales.
* Promotional campaigns increase overall sales volume.
* Foot traffic directly affects product sales.
* Seasonal products show higher demand during peak periods.

## Tools Used

* Tableau
* Data Visualization
* Retail Sales Dataset
